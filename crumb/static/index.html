The attack Crumb is built to catch

A poisoned tool drives an agent into a call the human never authorized. In a controlled lab this hijacked 5 of 7 models into an unauthorized data export. Crumb can’t block it — it’s a flight recorder, not enforcement — but it proves, per call, the human didn’t direct it.

Hash-chained + Ed25519-signed, with the Merkle root anchored to Sigstore's public Rekor log. A careless edit breaks the chain; a key-holder who re-signs a rewritten history is caught by the anchor instead. Don't trust this page. Verify it yourself: python -m crumb.verify and the Rekor link above.
